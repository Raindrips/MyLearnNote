## rust 语言

## rust的语言特性

1. rust 是一个争对于安全性的语言而发明的
2. rust 是一个编译型语言,和c++类似,可以直接编译成可执行文件
3. rust 相比c++有更好的语法设计,能在编译期间进行内存边界检查,有语法所有权的概念,有内存错误的检查,更人性化的错误位置显示

## rust 的语法

文件后缀是`.rs`

每一条语句以`;`结尾

### 打印输出语句

`println!` 打印输出的宏函数,并会自动换行

`print!`打印输出,不换行



### 编译rust文件

```sh
## 安装好工具链,rustc 文件名
rustc main.rs
```
