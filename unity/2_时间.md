## Time

获取时间信息的接口。帧率：每秒钟绘制的次数，次数越高表现的越精细。

| 静态变量   |                                                              |
| ---------- | ------------------------------------------------------------ |
| deltaTime  | 以秒计算，完成一帧的时间（只读）。                           |
| frameCount | 已经传递帧的总数（只读）。                                   |
| time       | 以秒计算，从游戏开始的时间（只读）。也就是说，从游戏开始到到现在所用的时间 |
| timeScale  | 时间的缩放。这可以用于减慢运动效果。                         |
