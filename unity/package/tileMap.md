# 2D TileMap

## 下载2d地图编辑器

+ 进入 Windows ->  Package Manager	
+ 找到 2D TileMap Edit 然后下载



## 创建2d地图

+ create ->2d object -> tilemap
+ 切换到2d模式进行预览

### Gird组件

参数信息

+ cell Size 表格大小
+ Cell Gap 位置间距
+ Layout 布局方式
	+ Rectangle 矩形
	+ Hexagon 六边形 
	+ Isometric 斜面
+ Cell Swizzle 表格方向

### Tilemap地图

如果要进行TileMap地图编辑, 进入Windows -> Tile Palette 将编辑工具打开,然后进行编辑
