# coco2d-x

## 概述

Cocos2d-x 是一个游戏引擎,内部封装了游戏的常用引擎接口,代码风格充分的利用了面向对象编程的特性,使用户在使用的过程非常方便和轻松

## 常见数据结构

###  二维向量

类名:`Vec2`

里面实现了二维坐标系常用函数,使得使用起来非常轻松

```cpp
class Vec2{
    float x;
    float y;
	Vec2();
	Vec2(float x,float y);
    
}
```
