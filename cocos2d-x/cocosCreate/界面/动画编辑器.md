# 动画编辑器

## 工具的使用

1. 选择要添加动画组件的节点
2. 添加动画组件
3. 编辑动画资源
4. 在组件中加载动画资源文件

## Clip组件

+ Default 默认的动画组件
+ Clips 通过脚本访问的组件数量
+ Play onLoad 加载时自动播放默认组件

## cc.Animation

```js
//获取动画组件
let anim=this.node.getCompont( cc.Animation);

//播放动画
anim.play('动画名称');


//叠加播放,不会关闭当前的动画
anim.playAdditive();
//停止动画
anim.stop();

//暂停动画
anim.pause();

//属性
anim.defaultClip;			//默认动画
anim.currentClip;			//当前播放的动画
anim.lastFrame;				//播放帧

//播放事件
anim.on('动画',()=>{},this);
```
