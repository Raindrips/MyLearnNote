## 委托类

委托类是cocos2d的入口文件里面有程序调用的生命周期

```cpp
class AppDelegate{
    void initContextAttrs();
    //启动完成后调用的函数
    bool applicationDidFinishLaunching();
    //进入后台时候调用的函数
    void applicationDidEnterBackground();
    //进入前台时候调用的函数
    void applicationWillEnterForeground();
};
```

# 场景节点

场景，节点，节点文件的区别

- 场景文件不能改变大小
- 图层文件可以改变大小
- 节点文件没有大小



- 场景文件不能与场景文件嵌套
- 图层可以和节点嵌套
