### 网格特效的原理

#### 网格特效的使用

```mermaid
graph TB
A[]
B[]
A[] ---> B[]
```



#### 网格特效的使用方法

网格特效继承了Action 类,可以用`runAction`来使用

```cpp
class GridAction{
    virtual GridBase* getGrid;

    virtual void setGrid(GridBase* grid);

    Action *runAction(Action *action);
}
```
