## 包装类

Value,将常用数据类型进行包装成类来使用,除了  `int` `char` `float` ,还包括了c++标准类

+ `std::string`
+ `std::vector<Value>`
+ `std::unordered_map<string,Value>`
+ `std::map<string,Value>`

```cpp
class Value{
	Value();						//参数,包装对象
    Value asValue();				//获取对应类型数据
  	bool isNull();					//判断是否为空
    string getDescription(); 		//获取对象信息
};
```
