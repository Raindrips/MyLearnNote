# 数据交换格式

常用文档格式

+ xml
+ json
+ csv

## Xml

默认utf-8编码

```xml
<?xml version="1.0" encoding="utf-8"?>

<node>
	<tag id="id">content</tag>
</node>
```

## json

```
{
	a:10,
	b:3.14,
	c:"hello world"
}
```

## csv

`csv`默认是`ANSI`编码

```csv
1,"name",2020/12/11,id
2,"name2",2020/12/12,id2
```
