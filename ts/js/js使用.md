# Javascirpt

js是一个解释型的编程语音,通过一个程序来进行执行

## 开发环境搭 建

1. 安装[nodejs][https://nodejs.cn]



## 使用js

```js
var num;			//定义变量
num=8.5;			//变量赋值
//输出变量的值
console.log(num);	

//变量的类型
console.log(typeof num);

//变量的运算
num=num+1;
```

## 表达式

```js
var a;
a=1;	//赋值语句
a=a-1;	//减去一个数	

a==a;  //比较运输符
//小数尽量不要用于比较
```
